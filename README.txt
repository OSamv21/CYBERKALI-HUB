


##  Overview

**CyberKali Hub** is an interactive web-based terminal simulator designed to help cybersecurity students and enthusiasts learn Kali Linux commands in a safe, simulated environment. With over **200+ commands** across 6 categories, this tool provides detailed explanations, usage examples, and a fully functional terminal simulator without executing any real system commands.

##  Features

###  **Core Features**
- **Interactive Terminal Simulator**: Full-featured terminal with command history, tab completion, and realistic outputs
- **200+ Kali Linux Commands**: Comprehensive database covering Basic, Intermediate, Advanced, Expert, and Dangerous commands
- **Smart Search**: Real-time command search with filtering by category/keyword
- **Command Details**: Click any command for detailed usage, examples, and warnings
- **Click-to-Copy**: One-click copy commands to clipboard
- **Category System**: Commands organized by difficulty and purpose

###  **Safety Features**
- **100% Safe Environment**: No real commands are executed
- **Dangerous Command Warnings**: Clear warnings for destructive commands
- **Educational Focus**: Designed for learning and practice

###  **UI/UX Features**
- **Cyberpunk Aesthetic**: Neon green/blue theme with matrix effects
- **Responsive Design**: Works on desktop and mobile
- **Interactive Animations**: Smooth transitions and visual feedback
- **Command Statistics**: Dashboard with command counts and categories
